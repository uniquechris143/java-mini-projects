# ☕ Java Foundation Projects

A collection of beginner-friendly Java programs focused on data handling, user input, randomization, and formatted console output. These projects demonstrate skills in working with different data types, structured program flow, and console-based UI design.

---

## 📁 Included Projects

---

### 1. COC Enrollment Receipt (`COCEnrollmentReceipt.java`)
This program simulates a school enrollment system by capturing user input and generating a structured receipt.

**Key Skills:**
- User interaction using `Scanner`
- Handling different data types (`String`, `double`, `char`)
- Character input using `scanner.next().charAt(0)`
- Basic receipt formatting in console output

---

### 2. Student Status Board (`StudentStatusBoard.java`)
A simple console-based dashboard that displays structured student information in a clean format.

**Key Skills:**
- Variable initialization and data management
- Use of primitive and non-primitive data types
- Console formatting and layout design
- Improving readability using symbols and spacing

---

### 3. Billing System (`BillingSystem.java`)
A simple billing program that applies a discount if the user is a student.

**Key Skills:**
- Boolean logic and conditionals (`if-else`)
- Parsing user input to boolean (`Boolean.parseBoolean`)
- Basic discount calculation
- Compound assignment operators (`*=`)
- Console input validation

**How it works:**
- User enters total amount
- User inputs whether they are a student (true/false)
- If true → 10% discount is applied
- Otherwise → full price is shown

---

### 4. Monster Battle (`MonsterBattle.java`)
A simple turn-based console battle system where the player fights a randomly generated monster.

**Key Skills:**
- Random number generation (`Random`)
- Game logic using conditionals
- Floating-point math operations
- Type casting and formatting output
- Basic combat simulation logic

**How it works:**
- A monster with random HP (50–100) is generated
- Player enters attack power
- If attack ≥ HP → monster is defeated
- Otherwise → remaining HP is shown
